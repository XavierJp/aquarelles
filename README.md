## Quick Start

```bash
# Backend
cd backend
npm install
npm start

# Frontend  
cd frontend
npm install
npm start
```

Visit `http://localhost:3000` to explore podcasts on the map.

## Tech Stack

- **Frontend**: React, Leaflet.js, CSS3
- **Backend**: Node.js, Express
- **Map**: Stamen Design watercolor tiles via Cooper Hewitt
- **Data**: File-based storage (JSON)
